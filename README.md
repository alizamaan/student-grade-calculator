# Student Marks Calculator 
A Simple **Student Marks Calculator** using HTML,CSS and JavaScript that calculates **Grade and Percentage**.
## Features
- Enter marks for 5 subjects : Maths, Science, English, Computer, Hindi,
- Calculates **Percentage**
- Displays **Grade**(A+,A,B,C,Fail) based on percentage
- Simple and beginner-friendly UI
## Skills Used
- HTML
- CSS
- JavaScript
## How to use
- Open calculator.html in your browser
- Enter marks for each subject
- Click **Calculate Result**
- View your **Percentage and Grade**
